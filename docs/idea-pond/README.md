# Idea Pond

Documents for ideas that are **not yet assigned to an epic** — the "someday /
maybe" pool. An idea can be documented here (design musings, references, sketches,
audio notes) even when it has no epic and only minimal planning; that's exactly
what the single-page `doc:1` marker on a SprintBoard card signals.

When an idea graduates into an epic, move its docs into `docs/epics/<E#>/` and
update any `docRef` on its ticket.

See `.claude/skills/sprintboard-usage/SKILL.md` for the full workflow.
